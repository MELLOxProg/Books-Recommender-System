## Application Overview

The Book Recommender System provides a clean, intuitive interface for discovering book recommendations. Built with Streamlit, the application offers a seamless experience for exploring personalized book suggestions.

### Launching the Application

Once you have the system installed, start the application:

```bash path=null start=null
streamlit run app.py
```

The application will open in your default browser at `http://localhost:8501`.

## üö® Troubleshooting Common Issues

<AccordionGroup>
  <Accordion title="‚ùå No Recommendations Appearing">
    **Possible Causes:**
    - Book not found in filtered dataset
    - Model files missing or corrupted
    - Network issues loading cover images
    
    **Solutions:**
    - Try a different, more popular book
    - Check console for error messages
    - Refresh the page and try again
    - Verify model files exist in artifacts/ directory
  </Accordion>

  <Accordion title="üñºÔ∏è Missing Book Covers">
    **Possible Causes:**
    - External image URLs are broken
    - Network connectivity issues
    - Image hosting service unavailable
    
    **Solutions:**
    - Wait for images to load (may take time)
    - Check your internet connection
    - Book titles still appear even without covers
    - Try refreshing the page
  </Accordion>

  <Accordion title="üîÑ Dropdown Not Loading">
    **Possible Causes:**
    - Large dataset taking time to load
    - Memory issues with browser
    - JavaScript errors
    
    **Solutions:**
    - Wait for initial loading to complete
    - Refresh the browser page
    - Try a different browser
    - Check browser console for errors
  </Accordion>

  <Accordion title="‚ö° Slow Performance">
    **Possible Causes:**
    - First-time model loading
    - Large dataset processing
    - Limited system resources
    
    **Solutions:**
    - First recommendation is slower (model loading)
    - Subsequent recommendations are faster
    - Close other browser tabs to free memory
    - Ensure adequate system RAM (8GB+ recommended)
  </Accordion>
</AccordionGroup>
