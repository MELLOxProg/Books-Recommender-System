## Development Environment Setup

This guide will help you set up a complete development environment for the Book Recommender System.

## Prerequisites

Before starting, ensure you have the following installed:

- **Python 3.7+** (3.9+ recommended)
- **Git** for version control
- **Jupyter Notebook** for ML development
- **pip** or **conda** for package management

## Installation Methods

### Method 1: Using pip (Recommended)

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/MELLOxProg/Books-Recommender-System.git
    cd Books-Recommender-System
    ```
  </Step>

  <Step title="Create Virtual Environment">
    ```bash
    # Create virtual environment
    python -m venv book-recommender-env
    
    # Activate environment
    # Windows
    book-recommender-env\Scripts\activate
    # macOS/Linux
    source book-recommender-env/bin/activate
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    # Install basic dependencies
    pip install -r requirements.txt
    
    # Install development dependencies
    pip install jupyter matplotlib seaborn pandas scikit-learn
    ```
  </Step>

  <Step title="Install Local Package">
    ```bash
    # Install the local package in editable mode
    pip install -e .
    ```
  </Step>
</Steps>

### Method 2: Using conda

<Steps>
  <Step title="Create Conda Environment">
    ```bash
    conda create -n book-recommender python=3.9
    conda activate book-recommender
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    # Install core packages
    conda install numpy pandas scikit-learn
    
    # Install Streamlit
    pip install streamlit
    
    # Install development tools
    conda install jupyter matplotlib seaborn
    ```
  </Step>
</Steps>

## Data Setup

### Download Dataset

The system uses the Book Crossing dataset. You have two options:

<AccordionGroup>
  <Accordion title="Option 1: Use Existing Data">
    If the `data/` directory already contains the CSV files, you're ready to proceed.
  </Accordion>

  <Accordion title="Option 2: Download from Kaggle">
    If you need to download the dataset:
    1. Visit the [Book Crossing Dataset](https://www.kaggle.com/ra4u12/bookrecommendation)
    2. Download the three CSV files
    3. Place them in the `data/` directory:
       - `BX-Books.csv`
       - `BX-Users.csv`
       - `BX-Book-Ratings.csv`
  </Accordion>
</AccordionGroup>

### Verify Data Files

```bash
# Check if data files exist
ls data/
# Should show: BX-Books.csv, BX-Users.csv, BX-Book-Ratings.csv
```

## Model Generation

### Generate Model Artifacts

If model artifacts don't exist, you need to generate them:

<Steps>
  <Step title="Start Jupyter Notebook">
    ```bash
    jupyter notebook "Books Recommender.ipynb"
    ```
  </Step>

  <Step title="Run All Cells">
    Execute all cells in the notebook to:
    - Load and clean the dataset
    - Create user-item matrices
    - Train the k-NN model
    - Generate model artifacts
  </Step>

  <Step title="Verify Artifacts">
    ```bash
    ls artifacts/
    # Should show: model.pkl, book_names.pkl, book_pivot.pkl, final_rating.pkl
    ```
  </Step>
</Steps>

## Configuration

### Streamlit Configuration

The system automatically creates Streamlit configuration, but you can customize it:

```bash
# Create custom Streamlit config
mkdir -p ~/.streamlit
cat > ~/.streamlit/config.toml << EOF
[server]
port = 8501
enableCORS = false
headless = true

[theme]
primaryColor = "#0D9373"
backgroundColor = "#FFFFFF"
secondaryBackgroundColor = "#F0F2F6"
textColor = "#262730"
EOF
```

### Environment Variables

Create a `.env` file for custom configurations:

```bash
# .env file
DEBUG=true
MODEL_PATH=artifacts/
DATA_PATH=data/
LOG_LEVEL=INFO
```

## Testing the Setup

### Run the Application

```bash
# Start the Streamlit application
streamlit run app.py
```

The application should start and open in your browser at `http://localhost:8501`.

### Verify Functionality

<Steps>
  <Step title="Check Model Loading">
    The application should load without errors and display the interface.
  </Step>

  <Step title="Test Recommendations">
    Select a book from the dropdown and click "Show Recommendation".
  </Step>

  <Step title="Verify Results">
    You should see 5 book recommendations with cover images.
  </Step>
</Steps>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="ModuleNotFoundError">
    **Problem**: Missing dependencies
    **Solution**: 
    ```bash
    pip install -r requirements.txt
    pip install -e .
    ```
  </Accordion>

  <Accordion title="FileNotFoundError: artifacts/">
    **Problem**: Model artifacts missing
    **Solution**: Run the Jupyter notebook to generate artifacts
  </Accordion>

  <Accordion title="Memory Errors">
    **Problem**: Insufficient memory for large datasets
    **Solution**: 
    - Close other applications
    - Use data type optimization
    - Consider using a machine with more RAM
  </Accordion>

  <Accordion title="Port Already in Use">
    **Problem**: Port 8501 is occupied
    **Solution**: 
    ```bash
    # Use different port
    streamlit run app.py --server.port 8502
    ```
  </Accordion>
</AccordionGroup>
