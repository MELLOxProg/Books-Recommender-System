## ðŸš€ Installation

<Steps>
  <Step title="Clone the Repository">
    ```bash path=null start=null
    git clone https://github.com/MELLOxProg/Books-Recommender-System.git
    cd Books-Recommender-System
    ```
  </Step>

  <Step title="Create Virtual Environment (Recommended)">
    <Tabs>
      <Tab title="Using venv">
        ```bash path=null start=null
        # Create virtual environment
        python -m venv book-recommender

        # Activate (Windows)
        book-recommender\Scripts\activate

        # Activate (macOS/Linux)
        source book-recommender/bin/activate
        ```
      </Tab>
      <Tab title="Using conda">
        ```bash path=null start=null
        # Create conda environment
        conda create -n book-recommender python=3.9

        # Activate environment
        conda activate book-recommender
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Install Dependencies">
    ```bash path=null start=null
    pip install -r requirements.txt
    ```
    
    This installs:
    - `streamlit` - Web application framework
    - `numpy` - Numerical computing library
    - Local package dependencies in editable mode
  </Step>

  <Step title="Verify Model Files">
    Check that the pre-trained model artifacts are present:
    ```bash path=null start=null
    # List artifacts directory
    ls artifacts/
    # Should show: model.pkl, book_names.pkl, book_pivot.pkl, final_rating.pkl
    ```
    
    <Warning>
      **Model files missing?** If the artifacts folder is empty, you'll need to run the Jupyter notebook to generate the model files first. See the [Model Training](/model-training) guide.
    </Warning>
  </Step>

  <Step title="Launch the Application">
    ```bash path=null start=null
    streamlit run app.py
    ```
    
    The application will automatically open in your default browser at `http://localhost:8501`
    
    <Tip>
      If the browser doesn't open automatically, manually navigate to `http://localhost:8501`
    </Tip>
  </Step>
</Steps>

## ðŸ”§ Troubleshooting

<AccordionGroup>
  <Accordion title="ModuleNotFoundError: No module named 'src'">
    **Problem**: The local package isn't installed properly.
    
    **Solution**: Install the package in editable mode:
    ```bash path=null start=null
    pip install -e .
    ```
  </Accordion>

  <Accordion title="FileNotFoundError: artifacts/model.pkl">
    **Problem**: Model files are missing from the artifacts directory.
    
    **Solutions**:
    1. **Download pre-trained models** (if available):
       ```bash path=null start=null
       # Check if models are in a separate download
       ls -la artifacts/
       ```
    
    2. **Generate models from scratch**:
       ```bash path=null start=null
       # Run the Jupyter notebook
       jupyter notebook "Books Recommender.ipynb"
       # Execute all cells to generate pickle files
       ```
    
    3. **Verify data files exist**:
       ```bash path=null start=null
       ls data/
       # Should show: BX-Books.csv, BX-Users.csv, BX-Book-Ratings.csv
       ```
  </Accordion>

  <Accordion title="Streamlit app won't start">
    **Problem**: Streamlit or dependencies not installed correctly.
    
    **Solution**: Reinstall dependencies:
    ```bash path=null start=null
    pip install --upgrade streamlit numpy pandas
    streamlit run app.py
    ```
    
    **Alternative**: Check Streamlit version compatibility:
    ```bash path=null start=null
    streamlit --version
    pip install streamlit==1.28.0  # Use stable version
    ```
  </Accordion>

  <Accordion title="Empty recommendations or errors">
    **Problem**: Corrupted or missing model files, or data processing issues.
    
    **Solutions**:
    1. **Clear and regenerate models**:
       ```bash path=null start=null
       rm -rf artifacts/*.pkl
       jupyter notebook "Books Recommender.ipynb"
       ```
    
    2. **Check data integrity**:
       ```bash path=null start=null
       python -c "import pandas as pd; print(pd.read_csv('data/BX-Books.csv', sep=';', encoding='latin-1').shape)"
       ```
    
    3. **Verify pickle files**:
       ```bash path=null start=null
       python -c "import pickle; print([f for f in ['model.pkl', 'book_names.pkl'] if pickle.load(open(f'artifacts/{f}', 'rb'))])"
       ```
  </Accordion>

  <Accordion title="Port 8501 already in use">
    **Problem**: Another Streamlit app is using the default port.
    
    **Solution**: Use a different port:
    ```bash path=null start=null
    streamlit run app.py --server.port 8502
    ```
  </Accordion>
</AccordionGroup>
